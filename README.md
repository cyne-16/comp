## üé• Adding Your YouTube Video:

1. Open `computer_hardware.html` in a text editor
2. Find this section (around line 550):
```html
<!-- <iframe src="https://www.youtube.com/embed/YOUR_VIDEO_ID" ... -->
```
3. Replace `YOUR_VIDEO_ID` with your actual YouTube video ID
   - Example: If your URL is `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
   - Your video ID is: `dQw4w9WgXcQ`
4. Remove the `<!--` and `-->` comment tags
5. Save the file
